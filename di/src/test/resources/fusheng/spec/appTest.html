例子1：使用 ContainedApp

构建一个简单的MyApp，代码如下，其中有一个可注入的构造方法以及一个依赖 Wheel，同时，main 方法被 @Main 注解标注，作为容器app的入口点 

    @Getter
    @Singleton
    public class MyApp {

        private String result;
        private final Wheel wheel;

        @Inject
        public MyApp(@WheelType(type = SMALL) Wheel wheel) {
            this.wheel = wheel;
        }

        @Main
        public void main(String prefix) {
            result = String.format("%s %s", prefix, wheel.getType());
        }
    }
创建一个新的 ContainedApp， 然后 运行App，其启动参数设置为 Hello ，最终 MyApp 中的 result 正确设置为 Hello Small
